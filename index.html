    Welcome to Flow!

    This is a new programming language made in Python that resembles BASIC. It is new right now, but it will grow to help new programmers learn the basics of advanced programming languages, such as Java and C++. The syntax is similar to modern languages, yet it has many differences.

    Example File:

    ;This program takes an input and multiplies it by two.
in "What is your favorite number?" @a
echo "I'm going to multiply it by two!"
@a = 2 * @a
echo "Here is the result:"
echo @a
