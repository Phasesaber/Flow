    Welcome to Flow!

    This is a new programming language made in Python that resembles BASIC. It is pretty new right now, but will grow to be able to make windows, GUIs, and maybe even games!

    Example File:

    in "What is your favorite number?" @a
echo "I'm going to multiply it by two!"
@a = 2 * @a
echo "Here is the result:"
echo @a
